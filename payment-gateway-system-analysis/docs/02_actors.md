# Участники системы / System Actors

## Merchant / Мерчант

Интернет-магазин, который создает платежи и получает уведомления о результате оплаты.

An online store that creates payments and receives notifications about payment results.

## Customer / Покупатель

Пользователь, который оплачивает заказ на стороне интернет-магазина.

A user who pays for an order on the online store side.

## Payment Gateway / Платежный шлюз

Учебная система, которая принимает запросы на создание платежей, хранит данные и обрабатывает статусы платежей.

An educational system that receives payment creation requests, stores data, and processes payment statuses.

## Mock Acquiring Bank / Заглушка банка-эквайера

Тестовый внешний сервис, который имитирует успешный или неуспешный результат оплаты.

A test external service that simulates successful or failed payment results.

## Webhook Receiver / Получатель webhook

Тестовая система мерчанта, которая принимает уведомления от платежного шлюза.

A test merchant system that receives notifications from the payment gateway.
